Software

I develop and maintain the Python/C++ package dfba for dynamic flux-balance analysis (DFBA) simulations. This is joint work with Moritz E. Beber at DTU Biosustain that forms part of the openCOBRA code base for constraint-based reconstruction and analysis of metabolic models. Other software and scripts for data processing can be accessed via GitLab
